# Step 00 — Python Basics

This step focuses on learning the minimum Python required for understanding diffusion models and image generation systems.

## Concepts Covered

* Running Python files
* Virtual environments
* Variables
* Lists
* Loops
* Functions

## Files

* `hello.py` → first Python program
* `variables.py` → understanding variables
* `lists.py` → understanding lists
* `loops.py` → understanding loops
* `functions.py` → understanding functions

## Goal

The goal is not to master Python completely, but to become comfortable enough to work with tensors, images, and neural networks later.
