# Step 01 — Images as Numbers

This step focuses on understanding how computers represent images using numerical arrays.

## Concepts Covered

* Grayscale images
* RGB images
* NumPy arrays
* Pixel intensity
* Tensor shapes
* Image slicing
* Random image generation
* Noise representation

## Files

* `show_image.py` → displaying grayscale and RGB images
* `random_noise.py` → generating random noisy images

## Key Learning

Images are not magic — they are structured collections of numbers stored inside tensors/arrays.
